Regra da cascata

A regra da cascata no css funciona como uma cascata real, o ultimo codigo escrito, é o que permanecerá no site.

Especificialidades

Espeficidade no css é quando um elemento é mais especificos que outros. exemplo: id é mais especifico que class, e class é mais especifico que tag. logo a alteração que vai permanecer, é a do id.

Combinadores

    .primeira h1

    Se um h1 estiver com a class "primeira", voce pode colocar ".primeira h1 {}" no css, para combinar uma classe e um titulo. Ou seja, a alteração sera somente no h1, mesmo se houver outros codigos nessa classe.

    .primeira > h1

    O sinal ">" é uma flecha no css, ela indica que vai selecionar o filho da divisão que voce criou. Ou seja, se houver uma div com um h1, dentro de outra div com tambem um h1, o h1 alterado seria o que esta na div de fora, por conta que ele seria filho dela

    .primeiro h1+h1

    neste exemplo é usado da seguinte maneira, serão estilizados apenas os h1 que possuem um h1 antes deles, ou seja, o primeiro nao seria estilizado por não ter nenhum h1 antes dele.

display

    display: block

    todo elemento que voce colocar na sua pagina, ocupará a largura de tela inteira. Suas propriedades são, width, height, padding, margin, border.

    os elementos padrão são: h1, p, div, section, main, header, nav, ul, ol.

    display: inline

    se voce tiver um elemento com display inline abaixo de outro com esse display, no site, eles irão ficar um do lado do outro. Suas propriedades são, padding, margin, border.

    Os elementos padrão são: a, strong, span, img, button, input, em.

Border, padding e margin

    Border

    adiciona uma borda a um elemento

    exemplo: border: 1px solid black;

    Padding

    padding: adiciona um espaço entre o conteudo e a borda.

    exemplo: padding: 10px;

    Margin

    margin: adiciona um espaço entre uma borda e outra.

    exemplo: 50xp auto

box-sizing: border-box

se usado com margin: 0 e padding: 0, globalmente, ele deixa tudo junto para ser mais facil de estilizar os outros elemtenos depois.
